# Employee Salary Analyzer

## Overview

Employee Salary Analyzer is a Python and Pandas based data analysis project that analyzes employee salary data and generates useful business insights.

## Features

- Highest Paid Employee Analysis
- Lowest Paid Employee Analysis
- Top 10 Highest Salaries
- Department Wise Analysis
- City Wise Analysis
- Gender Wise Analysis
- Attendance Analysis
- Experience Analysis
- Experience vs Salary Analysis
- Overall Statistics
- CSV Report Generation

## Technologies Used

- Python
- Pandas
- NumPy

## Dataset Columns

- EmployeeID
- Name
- Department
- Gender
- Age
- Experience
- City
- Salary
- Bonus
- Attendance

## Project Structure

```text
Employee_Salary_Analyzer/
│
├── data.py
├── main.py
├── employees.csv
├── employee_report.csv
└── README.md
```

## How to Run

Generate Dataset:

```bash
python data.py
```

Run Analysis:

```bash
python main.py
```

## Output

The project generates:

```text
employee_report.csv
```

containing analyzed employee data and statistics.
