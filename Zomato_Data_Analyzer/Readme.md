# 🍽️ Zomato Data Analyzer

A Pandas-based data analysis project that analyzes Zomato restaurant data to generate meaningful business insights related to ratings, popularity, cuisines, costs, online ordering, and customer engagement.

---

## 📌 Project Overview

This project analyzes restaurant data from Zomato using Python and Pandas.

The project performs:

* Dataset Analysis
* Rating Analysis
* Popularity Analysis
* Location Analysis
* Restaurant Type Analysis
* Cuisine Analysis
* Cost Analysis
* Online Order Analysis
* Table Booking Analysis
* Votes Analysis

---

Dataset Used:

## 📈 Sample Insights

* Online ordering restaurants generally receive higher customer engagement.
* Table booking restaurants tend to have higher average ratings.
* A few locations dominate the restaurant ecosystem.
* Customer votes play an important role in restaurant popularity.
* Certain cuisines and restaurant types are significantly more common than others.
