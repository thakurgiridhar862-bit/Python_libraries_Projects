# 🏏 IPL Data Analyzer

A Pandas-based data analysis project that analyzes IPL (Indian Premier League) match and ball-by-ball data to generate meaningful cricket insights.

---

## 📌 Project Overview

This project analyzes IPL data from 2008 to 2016 using Python and Pandas.

The project performs:

- Match Analysis
- Team Analysis
- Player of the Match Analysis
- Batting Analysis
- Bowling Analysis
- Dismissal Analysis
- Venue Analysis
- Season Analysis
- Overall IPL Statistics

---

## 📂 Dataset

Datasets Used:

- data.csv (Match Information)
- deliveries.csv (Ball-by-Ball Information)

Dataset Period:

- IPL 2008 – IPL 2016

## 📈 Sample Insights

- Mumbai Indians emerged as the most successful team.
- Virat Kohli became the highest run scorer.
- Chris Gayle hit the most sixes.
- Lasith Malinga took the most wickets.
- M Chinnaswamy Stadium hosted the highest number of matches.
